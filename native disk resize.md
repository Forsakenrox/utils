# Последовательно вводим все команды
```
Всё протестировано на дистрибутиве debian13 с настройками по умолчанию для диска без LVM
```
**`swapoff -a`** (надо удалить раздел со свапом, иначе расширение не удастся т.к. он идёт вконце текущего диска)

**`fdisk -l`** (и запоминаем **Disk**, **Id** и **Type** раздела который хотим удалить (**/dev/mapper/ol-root**, **8e**, **Linux LVM**))

**`fdisk /dev/sda`**

## Параметры внутри fdisk 
**`d`** (partition /dev/sda1 | отвечаем **5** (swap))
**`d`** (partition /dev/sda1 | отвечаем **2** (хрень какая то для разделов))
**`d`** (partition /dev/sda1 | отвечаем **1** (основной раздел))

**`n`** (partition /dev/sda1 | primary отвечаем **p** и номер такой же как был в предыдущем пункте (**1**) )

**First sector** - смотрим какой был **First sector** у диска из **`fdisk -l`** (**start**/**end**) и указываем его, но по умолчанию если это был последний раздел в системе его сектор уже будет указан нужно выбрать (**default**)

**Last sector** - выбираем **default**, уже будет указан в промпте (максимально возможный секстор)  

На вопрос удалить LVM2_member signature **отвечаем N**, иначе **сломается**

**`t`** (partition /dev/sda1 номер указываем тот же что раньше (**1**) и **id** тип файловой системы из `fdisk -l` (**83**) )  
**`a`** (partition /dev/sda1 делаем раздел загрузочным)  
**`w`** (записываем изменения)

## Параметры после записи изменений на диск (``w``)

**`resize2fs /dev/sda1`** (указываем Disk который был в результате fdisk -l)

**`df -h`**

после этого нужно будет вернуть swap в виде файла подкачки а не раздела

**`fallocate -l 2G /swapfile`**
**`chmod 600 /swapfile`**
**`mkswap /swapfile`**
**`swapon /swapfile`**

после этого нужно изменить файл /etc/fstab и стереть там строчку монтирования swap и добавить такую строку вместо неё:

**`/swapfile none swap sw 0 0`**
